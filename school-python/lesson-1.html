✏️ 🐍 Урок 0: Разбор частых ошибок в Python (+ как исправить)

📚 1. IndentationError: Проблемы с отступами

if True:
print("Ошибка")  # ❌ Нет отступа после двоеточия

Как исправить:

✅ Всегда делайте 4 пробела (или таб) после :
✅ Настройте редактор кода для автоматических отступов
✅ Проверяйте выравнивание в блоках if/for/def

📚 2. SyntaxError: Ошибки синтаксиса

if x = 10:  # ❌ = вместо ==
print("Ошибка")

Как исправить:

✅ Сравнение: ==, присваивание: =
✅ Закрывайте все кавычки и скобки: "Hello") → "Hello"
✅ Проверяйте двоеточия в конце условий: if x == 10:

📚 3. NameError: Неизвестные переменные

print(age)  # ❌ Переменная не объявлена

Как исправить:

✅ Объявляйте переменные перед использованием:
age = 25  # ← Добавить это
print(age)

✅ Проверяйте регистр: Name ≠ name
✅ Импортируйте нужные модули

📚 4. TypeError: Несовместимые типы

num = input("Введите число: ")  # ❌ Возвращает строку
result = num + 5  # TypeError

Как исправить:

✅ Преобразуйте типы:
num = int(input("Введите число: "))  # ← Правильный вариант

✅ Проверяйте типы: type(num)
✅ Используйте явное преобразование: int(), str(), float()

✏️ 💡 Профилактика ошибок

  1. Используйте IDE с подсветкой синтаксиса (PyCharm, VSCode)
  2. Читайте сообщения об ошибках — они часто прямо указывают на проблему
  3. Печатайте код руками вместо копирования — лучше запоминается синтаксис
  4. Разбивайте код на части — тестируйте каждую функцию отдельно

✏️ 🐍 Урок 1: print(), input() вкратце переменные

Комментарии:

#комментарий к коду

1). Принт:

Команда print() выводит текст в консоль в скобках
И еще: если в скобках только текст он должен быть в кавчках пример: print("text"), но если надо вывести значение переменной то тогда без кавычек: print(название_переменной)
А ещё: в принт можно указать несколько параметров: print("значение 1","значение 2", название_переменной)

2). Вкратце переменные:

Переменные как бы ящик в который можно сложить что нибудь
Переменные создаются так:
название_перменной = "текст"
#или
название_переменной = 1 #можно просто цифру
После создания переменная будет как отдельная функция пример: print(название_переменной)
Частые ошибки: 1. Переменная называться так как какая нибудь команда 2. Название содержит пробел вместо него можно использовать "_" 3. Они чувствительны к регистру то есть "A" и "a" это разные переменные 4. Начинаться на цифру

3). Инпуты:

Команда input() выводит текст в скобках (там действуют такие же правила как и с принтом) в консоль и после него можно что нибудь написать и оно будет куда-то записываться пример:
Название_переменной = input("введите текст")
И в переменную название_переменной будет записываться то что мы написали

Пример кода:

MyVar1 = "" #создаем пустую  переменную
MyVar1 = input("введи параметр 1") # присваивается значение переменной MyVar1 значение через инпут
print("значение переменной MyVar1:" , MyVar1)

Выводит:

введи параметр 1: привет
значение переменной MyVar1: привет

✏️ 📘 Урок 2: Типы данных и базовые операции в Python

📚 1. Типы данных: что хранится в переменных?

⦁ Основные типы:

  • int — целые числа (пример: age = 15).
  • float — числа с плавающей точкой (пример: price = 99.9).
  • str — строки (текст в кавычках: name = "Анна").
  • bool — логические значения (True/False).

⦁ Как проверить тип?

Используйте type():

print(type(10))  # Выведет: <class 'int'>

📚 2. Операции с числами

⦁ Арифметика:

a = 10 + 5   # Сложение (15)
b = 10 - 2.5 # Вычитание (7.5)
c = 3 * 4    # Умножение (12)
d = 10 / 3   # Деление (3.333...)
e = 10 // 3  # Целочисленное деление (3)
f = 10 % 3   # Остаток от деления (1)
g = 2 ** 3   # Возведение в степень (8)

⦁ Приоритет операций:

Скобки () > Степень > Умножение/деление * / // % > Сложение/вычитание + -.

📚 3. Работа со строками

⦁ Конкатенация (сложение строк):

name = "Иван"
greeting = "Привет, " + name  # Результат: "Привет, Иван"

⦁ Умножение строк:

line = "-" * 20  # Создаст строку "--------------------"

📚 4. Преобразование типов

⦁ Примеры:

number_str = "10"
number_int = int(number_str)  # Преобразует строку в число (10)

age = input("Сколько вам лет? ")  # Ввод всегда возвращает строку!
age_int = int(age)  # Преобразование строки в целое число

⦁ Опасные моменты:

Ошибка при попытке преобразовать текст в число:

int("десять")  # Вызовет ошибку ValueError!
