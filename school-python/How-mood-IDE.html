Настройка IDE для Python

1. Visual Studio Code

1. Установите VSCode и python

2. Откройте расширения (Ctrl+Shift+X) → установите Python и Pylance

3. Настройте интерпретатор: Ctrl+Shift+P → Python: Select Interpreter

2. PyDroid3 (Android)

1. Установите из Google Play

2. Откройте меню → Интерпретатор → выберите версию Python

3. Включите подсветку синтаксиса в настройках редактора

СОВЕТ: когда первый раз запускаете программу он там опрос даст его вопросы можно скпнуть но в конце будут 3 галочки их лучше поставить

3. Pythonista (iOS)

1. Скачайте из App Store

2. Создайте новый скрипт: "+" → New Script

3. Для работы с библиотеками используйте встроенный менеджер пакетов

4. Настройте тему в Settings → Editor Theme
